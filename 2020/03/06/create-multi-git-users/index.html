创建多个Git用户、推送多个远程仓库

创建多个Git用户

  1. 生成多个ssh-key，并将对应的publicKey添加到git管理仓库中
  2. 配置~/.ssh/config
  3. 复制完整的 SSH 公钥并在对应的远程仓库中添加 SSH key 配置。
  4. 验证 SSH 连接

推送项目到不同仓库地址

  1. 进入项目，添加远程仓库B地址
